    What is Pychron

    Pychron is python package for data acquisition and processing of Ar-Ar analyses. The main Pychron code-base can be built into multiple application flavors to perform various tasks. For example, PyExperiment is the flavor used for data acquisition. PyCO2/PyDiode are flavors used for interacting with laser systems.
